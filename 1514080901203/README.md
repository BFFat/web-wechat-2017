# 小程序名称：快递一查

![image](https://github.com/cs2JeX/web-wechat-2017/blob/master/203/erweima.jpg)
## 小程序创意说明
####  这是一个可随时随地进行快递物流情况查询的小程序。其设计简洁，提供了快递查询，差错检测和查询历史记录等功能。
## 小程序页面设计
### 该小程序一共有两个页面：
####   页面一为查询页面，由快递公司选择框，快递号输入框，查询按钮组成。提供了查询，是否为空检测，报错等功能。查询功能主要通过服务器上的php接口完成对http请求向https请求的转换。
####   页面二为历史记录页面，当一个快递号查询成功时，会被自动记录到该页，以便重复查询时使用。页面下方提供了清除记录按钮，可对历史记录进行清除。该页主要通过微信小程序的缓存功能，对本地缓存进行增加和删除，从而实现历史记录功能。
