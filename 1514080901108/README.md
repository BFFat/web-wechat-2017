# 小程序--象棋棋谱

## 名称：
象棋棋谱

## 二维码：
![image](https://github.com/yuezhudxw/xiangqi/blob/master/picture/xiangqi.jpg)

## 创意说明：
 小程序有两个界面，分别是主页面，对战界面，实现象棋爱好者的下棋。

## 小程序页面设计：
小程序的各个页面之间是相互联系的，在主界面通过点击不同的对战模式，玩家即可进入不同的玩法模式中，满足了玩家的下棋要求。
